# 📊 Retail Sales Trend Analysis (Python + Power BI)

## 📌 Project Overview
This project analyzes retail sales data to identify trends, seasonal patterns, and performance insights.  
Python was used for data cleaning, preprocessing, and exploratory data analysis (EDA), and Power BI was used to create interactive dashboards for visualization.

---

## 🎯 Objective
The objective of this project is to:
- Analyze retail sales trends over time
- Identify seasonal patterns and revenue fluctuations
- Perform data cleaning and preprocessing using Python
- Visualize key insights through interactive Power BI dashboards
- Support business decision-making with data-driven insights

---

- Jupyter Notebook
- Power BI

## 🧹 Data Preprocessing (Python)
The following steps were performed using Python:
- Handling missing values
- Data cleaning and formatting
- Converting date columns
- Creating new calculated columns
- Aggregating monthly and yearly sales
- Exploratory Data Analysis (EDA)
